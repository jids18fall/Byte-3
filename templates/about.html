Accountability: About This Data Set

This mobile dataset contains my location statistics in past two weeks. They are collected using the AWARE framework with App installed on my iOS device. The whole dataset contains mobile data of locations, visited locations, and activity recognition. We download the dataset from Google Cloud SQL, by using SELECT statements with time conversion functions FROM_UNIXTIME(timestamp/1000, '%Y-%m-%d %H:%i:%s') before the data cleaning.

Where did this data come from?

These data are collected using my own mobile device, and collected through the AWARE framework with storage at Google Cloud SQL. The dataset used in the analysis is downloaded from the Google Cloud SQL by exporting the results of SELECT statements to csv files on Oct 17, 2018. We use two tables locations and plugin_ios_activity_recognition in the analysis.

Who has access to it?

The original data source in the SQL database can only be accessed by myself. However, the exported csv files, i.e., the data used in the analysis are freely available at https://console.cloud.google.com/storage/browser/mobile-byte3-jiaand .
